Ruby Operator Precedence

Operator Is it a method? Description
[ ], [ ]= yes collection element getter and setter
** yes exponential operator
!, ~, +, - yes not, complement, unary plus, unary minus
*, /, % yes multiplication, division, modulation
+, - yes addition, subtraction
>>, << yes right shift, left shift
& yes bitwise and
|, ^ yes bitwise or, bitwise xor
<=, <, >, >= yes comparison: less than or equal to, less than, etc.
<=>, ==, ===, !=, ~=, !~ yes comparator, equality, and pattern matching
&& no logical and
|| no logical or
.., ... no inclusive range, exclusive range
? : no ternary operator
=, %=, /=, -=, +=, |=, &=, >>=, <<=, *=, &&=, ||=, **=, { } no assignment and assignment shortcuts; block delimiter
